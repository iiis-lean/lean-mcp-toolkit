# Documentation

This directory contains the public documentation for `lean-mcp-toolkit`.

## Sections

- [Usage](usage/README.md): server startup, HTTP API entrypoints, remote CLI, and local shell
- [Tool Catalog](tool_catalog/README.md): group-level catalog and navigation map
- [Tool Reference](tool_catalog/tool_reference.md): detailed tool tables, provenance, and backend notes
- [Architecture](architecture/README.md): implementation structure, plugin wiring, and backend/interface layers
- [Configuration](configuration/README.md): server, group activation, backend selection, and CLI defaults

## Documentation Scope

- `README.md` at the repository root is the entry page for new users.
- `docs/usage/` explains how to run and call the toolkit.
- `docs/tool_catalog/` documents what the toolkit exposes.
- `docs/architecture/` explains how the project is structured internally.
- `docs/configuration/` documents runtime and backend configuration.
